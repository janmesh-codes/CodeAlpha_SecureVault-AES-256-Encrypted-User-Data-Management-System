
# 🔐 SecureVault – AES-256 Encrypted User Data Management System

A secure full-stack web application developed as part of the **CodeAlpha Cyber Security Internship**. SecureVault demonstrates modern web security techniques, including SQL Injection prevention, AES-256 encryption, secure authentication, and role-based access control.

---

## 📌 Project Overview

SecureVault is designed to protect sensitive user information from common web vulnerabilities. It implements industry-standard security practices to ensure secure data storage and user authentication.

---

## ✨ Features

- 🔑 Secure User Registration & Login
- 🛡️ SQL Injection Detection & Prevention
- 🔒 AES-256 Encryption for Sensitive Data
- 🔐 Password Hashing using Secure Algorithms
- 👤 Role-Based Access Control (Admin/User)
- 📋 Security Activity Logging
- 📊 Admin Dashboard
- 📱 Responsive User Interface
- ☁️ Cloud Deployment Ready

## 🚀 Installation

### 1. Clone the repository

```bash
git clone https://github.com/janmesh-codes/CodeAlpha_SecureVault-AES-256-Encrypted-User-Data-Management-System.git
```

### 2. Navigate into the project

```bash
cd CodeAlpha_SecureVault-AES-256-Encrypted-User-Data-Management-System
```

### 3. Install dependencies

```bash
npm install
```

### 4. Start the development server

```bash
npm run dev
```
